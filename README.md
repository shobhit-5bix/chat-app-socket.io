# chat-app-socket.io

"npm i" to install the node_modules

about the chat app : 
  this web app is made with websocket -> socket.io which is used on client side as well as server side.
  you can broadcast messages to everyone or converse privately by joining same room.

  -> for now this app have no frontend or ui design and also need some correction at server side.

This app will be updated along with the implementation of following featues:
    -> clean and beautiful ui
    -> super secure encryption algorithm for text messages (exclusive feature)
    -> auth feature so that user can login from anywhere
    -> for now there will be no option to save your messages or data
